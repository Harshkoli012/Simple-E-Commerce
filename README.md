# Simple E-Commerce Shopping Cart

A clean, lightweight, and interactive shopping cart application built using vanilla frontend web technologies. Users can browse a product catalog, add items to their cart dynamically, see real-time price updates, and process a simulated checkout session.

## 🚀 Features
* **Dynamic Content Injection:** Product data is safely mapped from JavaScript arrays and rendered directly into the DOM structure upon initialization.
* **Live Calculation Pipeline:** Automatically tracks state mutations to calculate accurate subtotal prices instantly.
* **Contextual View States:** Seamlessly switches interface panels (like hiding the checkout container and displaying "Your cart is empty" notices) based on real-time array status lengths.
* **State Resets:** Instantly flushes internal arrays and cleans user screens when clearing or finishing transactions.

## 🛠️ Built With
* **HTML5:** Semantic content layouts and view containers.
* **CSS3:** Responsive structural design layouts and class utility triggers (like `.hidden`).
* **JavaScript (ES6+):** Advanced DOM manipulation, custom event tracking delegation, and modern functional array tracking.
